# 老刘四步选股框架

> 核心口诀：**排雷 → 估值 → 精选 → 量价**
> 
> 别贪多，10只精挑 2-3 只就够了。

---

## 第一步：三大硬指标排雷（3分钟初筛）

### 1. ROE：连续 5 年不低于 15%
- ROE = 净利润 / 净资产
- 连续5年 ≥ 15% 说明公司盈利能力稳定
- 低于 10% 的公司通常不值得长期持有

**本 skill 实现**：因免费 API 不直接提供 ROE，采用 **隐含 ROE ≈ PB / PE** 作为代理指标，并提示用户用 F10 年报复核。

### 2. 毛利率与净利率：看趋势
- 稳定或提升为佳
- 毛利率连续下滑 = 行业竞争加剧或产品竞争力下降
- 净利率波动大 = 盈利质量不稳定

**本 skill 实现**：标注为“需 F10 历史财务数据补充”，提醒用户人工复核。

### 3. 经营现金流：检验利润质量
- 经营现金流净额应持续为正
- 利润增长但现金流长期为负 → 警惕财务造假或回款问题

**本 skill 实现**：标注为“需 F10 现金流量表补充”。

---

## 第二步：PEG 估值参考

### 公式
```
PEG = PE / 盈利增速（%）
```

### 判断标准
| PEG 区间 | 含义 |
|---------|------|
| PEG < 1 | 相对便宜，性价比高 |
| 1 ≤ PEG < 2 | 合理区间 |
| PEG ≥ 2 | 偏贵，需确认高增速能否持续 |

### 高景气赛道特殊处理
- 高景气赛道 PE 贵一点正常
- 关键是 **增速跟不跟得上**
- 增速 > 30% 的赛道，PEG 可适度放宽到 1.5~2

**本 skill 实现**：PE 来自实时行情；盈利增速用近 60/120 日价格涨幅年化作为代理，并明确提示精确 PEG 需用净利润增速。

---

## 第三步：精挑细选（十中选三）

### 1. 业绩预期差：关注边际改善
- 环比改善信号比同比更重要
- 关注：营收增速拐点、净利润增速转正、毛利率回升

### 2. 股东人数 / 筹码集中度
- **低位筹码集中** = 有戏（主力在收集）
- **高位散户涌入** = 等等（筹码分散，风险大）

### 3. 相对强度
- 同板块中，选走势最强的 2~3 只
- 强者恒强是 A 股常见特征

**本 skill 实现**：
- 边际改善：用 MA20 拐头、站上 MA20/MA60 判断
- 筹码集中度：标注需 F10 股东户数数据
- 相对强度：用个股涨跌幅 vs 板块平均涨跌幅

---

## 第四步：成交量验证资金态度

### 健康信号
- **上涨温和放量 + 回调缩量** = 量价健康，资金持续流入
- **放量创新高** = 资金认可，突破有效

### 危险信号
- **放量滞涨** = 抛压重，主力可能在出货
- **缩量硬拉新高** = 拉升乏力，假突破风险
- **上涨缩量** = 动能不足，难以持续

**本 skill 实现**：
- 分离近 20 日上涨日与下跌日成交量
- 计算最新量比、是否创新高
- 输出量价健康度评分

---

## 综合评分规则

| 维度 | 权重 |
|------|------|
| 硬指标排雷 | 25% |
| PEG 估值 | 25% |
| 精挑细选 | 25% |
| 成交量验证 | 25% |

### 降级规则
任一维度得分 < 40 分，总分上限降至 60 分。

### 评级
| 综合得分 | 评级 | 建议 |
|---------|------|------|
| ≥ 80 | A 重点跟踪 | 加入观察池，等待买点 |
| 65 ~ 79 | B 符合条件 | 可结合技术面找买点 |
| 50 ~ 64 | C 一般 | 观望或等待确认 |
| < 50 | D 暂不考虑 | 放弃或等基本面改善 |

---

## 使用建议

1. 先用第一步排除明显有问题的公司
2. 再用第二步判断估值是否合理
3. 第三步从剩余标的中选相对强势、有改善信号的
4. 第四步用量价确认资金态度
5. **10只初筛 -> 3只精选 -> 2~3只建仓**，不要贪多

# 进阶策略：趋势动能 + 真实需求/小市值/财报验证

## 一、趋势动能系统（5/20日均线 + 周线MACD）

### 日线：5日线金叉20日线关注
- **金叉条件**：MA5 上穿 MA20
- **支撑条件**：价格紧贴 5/10 日线不破（允许 1% 以内虚破）
- **减仓信号**：5 日线下穿 20 日线

### 周线：MACD 零上金叉确认大趋势
- **关注信号**：周线 MACD 在零轴上方形成金叉
- **高胜率买点**：周线 MACD 零上金叉后的 **第 2 根红柱**
- **卖出信号**：周线 MACD 死叉
- **原则**：日线选时，周线定方向

**本 skill 实现**：
- `ma_cross_analysis()`：检测 5/20 日金叉死叉、5/10 日线支撑
- `weekly_macd_analysis()`：检测周线 MACD 零上金叉、红柱计数、死叉

---

## 二、真实需求与财报验证（9位数大佬思路）

### 1. 抛开噱头，只抓落地的真实需求
有效利好必须满足：
- 下游厂商已经下单采购
- 工厂排产排满
- 货品持续出货
- 现货价格上涨
- 上市公司财报主动提及新业务

**警惕**：只有概念吹牛皮、没有落地订单的新闻，全是炒作陷阱。

### 2. 利好换算成财报，小市值才有暴利空间
同样一笔产业链订单：
- 落到千亿巨头 → 营收涨幅微乎其微，股价难大涨
- 落到几十亿小盘公司 → 营收拉高两三成，直接改写全年业绩

**选股优先**：盘子小、主业专一、原有营收基数低、卡在产业链关键环节的冷门票。

**本 skill 实现**：
- 小市值评分：30~100 亿最佳，100~300 亿中等，300 亿以上弹性递减
- 估值弹性评分：低 PE + 小市值 = 修复空间大

### 3. 超额收益藏在认知误区里
- 很多牛股被市场贴错标签
- 市场认知与公司真实业务出现巨大落差 → 绝佳布局窗口

**本 skill 实现**：标注提醒“关注业务实质与市值错配的公司”，需人工/F10挖掘。

### 4. 用财报数据验证逻辑，杜绝纯讲故事
未来 1~4 个季度盯紧：
- 营收提速
- 毛利率抬升
- 在手订单增多
- 交货周期变长
- 产能开满
- 产品涨价
- 公司加大建厂投入

**原则**：数据兑现，逻辑成立；数据不及预期，果断离场。

**本 skill 实现**：所有财报验证项标注为“需 F10 数据人工复核”。

### 5. 分阶段把控仓位
| 阶段 | 仓位 | 条件 |
|------|------|------|
| 逻辑雏形 | 只观望 | 仅有初步逻辑，未确认 |
| 需求初现 | 20%~30% 试错 | 部分信号确认 |
| 财报落地 | 50%~70% 加仓 | 数据兑现 + 趋势动能共振 |
| 逻辑证伪 | 直接止损 | 数据不及预期或周线MACD死叉 |

---

## 综合选股框架 2.0

```
硬指标排雷 → PEG估值 → 精挑细选 → 量价验证
      ↓
趋势动能（5/20日线 + 周线MACD）
      ↓
真实需求/小市值/财报验证
      ↓
分阶段仓位管理
```

**本 skill 最终输出**：选股总分、选股评级、综合建议、仓位建议、四步得分、进阶信号明细。
