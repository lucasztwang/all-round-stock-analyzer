# 技术指标参考指南

## 移动均线 (MA)

| 周期 | 用途 | 判断标准 |
|------|------|---------|
| MA5  | 短线攻击线 | 价格在 MA5 之上为短线强势 |
| MA10 | 短线防守线 | 回踩不破可低吸 |
| MA20 | 中短线生命线 | 月线拐头向上是中期走好信号 |
| MA60 | 中期趋势线 | 季线定牛熊 |
| MA120 | 中长期半年线 | 牛熊分界线之一 |
| MA250 | 长期年线 | 牛熊分界线 |

**多头排列**：MA5 > MA10 > MA20 > MA60 → 强势上涨
**空头排列**：MA5 < MA10 < MA20 < MA60 → 弱势下跌

## MACD (指数平滑异同移动平均线)

默认参数：(12, 26, 9)

| 信号 | 含义 |
|------|------|
| DIF 上穿 DEA (金叉) | 买入信号 |
| DIF 下穿 DEA (死叉) | 卖出信号 |
| DIF/DEA 在零轴上方 | 多头市场 |
| DIF/DEA 在零轴下方 | 空头市场 |
| 柱状线由绿转红 | 动能转强 |
| 顶背离 | 价格新高、DIF 未新高 → 见顶 |
| 底背离 | 价格新低、DIF 未新低 → 见底 |

## RSI (相对强弱指标)

默认参数：14

| 区间 | 状态 | 操作 |
|------|------|------|
| RSI > 80 | 严重超买 | 减仓/观望 |
| 70 < RSI < 80 | 超买 | 注意风险 |
| 30 < RSI < 70 | 正常 | 持有/观望 |
| 20 < RSI < 30 | 超卖 | 关注反弹 |
| RSI < 20 | 严重超卖 | 可以分批低吸 |

## KDJ (随机指标)

默认参数：(9, 3, 3)

| 信号 | 含义 |
|------|------|
| K 上穿 D (金叉) | 买入信号 |
| K 下穿 D (死叉) | 卖出信号 |
| J > 100 | 超买，回调风险 |
| J < 0 | 超卖，反弹机会 |
| 低位金叉 (20以下) | 较强买入信号 |
| 高位死叉 (80以上) | 较强卖出信号 |

## 布林带 (BOLL)

默认参数：(20, 2)

| 位置 | 含义 |
|------|------|
| 价格触及上轨 | 短期超买，压力位 |
| 价格触及下轨 | 短期超卖，支撑位 |
| 带宽收窄 | 变盘信号 |
| 带宽扩张 | 趋势延续 |
| 价格沿上轨运行 | 强势上涨 |
| 价格沿下轨运行 | 弱势下跌 |

## 量价关系

| 模式 | 含义 |
|------|------|
| 放量上涨 | 多头强势，趋势健康 |
| 放量下跌 | 空头强势，注意风险 |
| 缩量上涨 | 动能不足，可能见顶 |
| 缩量下跌 | 抛压减轻，可能见底 |
| 天量天价 | 顶部信号 |
| 地量地价 | 底部信号 |

## 综合信号优先级

1. **趋势定方向** — 先看 MA 排列，确定大方向
2. **MACD 选时机** — 金叉/死叉确定买卖点
3. **RSI/KDJ 辅助** — 判断短期超买超卖
4. **量价验证** — 量能配合才可靠
5. **布林带定位** — 判断当前价格位置
